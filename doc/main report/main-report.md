# 总报告

## 问题及背景

为了帮助同学们更好地学习物理，更加形象地理解刚体的碰撞，本项目开发了一个模拟多刚体碰撞的图形界面，可以随心所欲地加入不同形状的刚体并且模拟他们碰撞的过程。

## 最终运行效果图

![final](../images & videos/final.png)

程序界面简洁，左侧为刚体所在的物理空间，右侧为操作栏。支持**创建任意多边形刚体**、**指定刚体的各初态量**、**开始/停止模拟**、**向空间添加力场**及**清屏**。所有操作都会在操作栏的迷你命令行得到反馈。
